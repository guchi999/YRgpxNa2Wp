　ヤマレコの山行記録、および「ヤマプラ」や「らくル～ト」で作られた計画のルートを地名付きGPXファイルとしてダウンロードしたとき、地名はトラック内のポイントの名前として入っており、カシミール3Dなどのソフトでは認識されません。それをウエイポイントに変換します。


　使い方は、変換するGPXファイルを画面にドロップするかファイル選択のボンタンで開き [変換したファイルを保存] をクリックするだけです。
　変換後のファイルはダウンロードフォルダに xxxxxx_wp.gpx の名前で保存されます。(xxxxxx は入力ファイルの名前)


　地名に [ふりがな] や「山のデータ」へのリンクがあれば、それをウエイポイントに含める/含めないはラジオボタンで選択できます。
　リンクは<desc>タグに、ふりがなは表示の有無にかかわらず<cmt>タグに入ります。
